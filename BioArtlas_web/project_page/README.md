# BioArtlas Project Page

[`project_page/`](.) is the landing-page frontend for the BioArtlas project.
It is separate from the interactive visualization app in the parent
[`../`](..).

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Scope

This app is intended for the static project page, summary content, and link
hub. It does not contain the reproduction pipeline or the main interactive
clustering visualization.
