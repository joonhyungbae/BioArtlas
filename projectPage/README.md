# BioArtlas Project Page

[`projectPage/`](.) is the landing-page frontend for the BioArtlas project.
It ships with this repository and is separate from the interactive
visualization app, which lives in another repository.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Scope

This app is intended for the static project page, summary content, and link
hub. It does not contain the reproduction pipeline or the main interactive
clustering visualization.
